# Ethos themes

All themes are published on the [Ethos-Feedback-Community repository](https://github.com/FrSkyRC/ETHOS-Feedback-Community/tree/26.1/lua/themes).

I also publish them here for convenience as you can find in the [release](https://github.com/flyingeek/ethos-themes/releases/latest) section each theme in a zip file.

You can try them in the VS Code simulator: clone or download this repository.

Install in VS Code:

- the [Ethos Simulator extension](https://marketplace.visualstudio.com/items?itemName=bsongis.ethos)
- the [Ethos-DevTools extension](https://marketplace.visualstudio.com/items?itemName=flyingeek.ethos-devtools)
- the [Lua extension](https://marketplace.visualstudio.com/items?itemName=sumneko.lua)

Then open the simulator by clicking at the bottom right on "X20PRO_FCC" and use the first option "Deploy & Launch Sim".

Now goto System > General > Theme and select the theme you want to try.
